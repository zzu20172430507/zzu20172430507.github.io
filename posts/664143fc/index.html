leetcode 398 随机数索引

2022-04-25 leetcode 0 0 评论 字数统计: 293(字) 阅读时长: 1(分)

题号：398

题目链接：https://leetcode-cn.com/problems/random-pick-index/

难度：中等

题意：给一个整数数组nums和一个target，随即输出给定目标数字target的索引。（假定给定的数字一定在数组中且可能由重复元素）要求实现一个solution类

题解：https://leetcode-cn.com/problems/random-pick-index/solution/sui-ji-shu-suo-yin-by-leetcode-solution-ofsq/

Solution 1：哈希（空间换时间）

存所有元素对应的下标，元素x对应下标为一个vector，从vector中随机返回一个数

Solution 2：水塘抽样（时间换空间）

如果数组以文件的形式存储，那么我们无法把所有的数存在内存里，因此要找一种空间复杂度更低的算法

在第i次遇到值为target的元素时，随机从[0,i）中选择一个整数，如果其等于0，则返回值为该元素的下标，否则返回值不变

class Solution {
vector &nums;
public:
Solution(vector &nums) : nums(nums) {} 

int pick(int target) {
    int ans;
    for (int i = 0, cnt = 0; i < nums.size(); ++i) {
        if (nums[i] == target) {
            ++cnt; // 第 cnt 次遇到 target
            if (rand() % cnt == 0) {
                ans = i;
            }
        }
    }
    return ans;
}

};
